# LCK 취케팅 매크로

LCK 암표상의 매크로를 대응하기 위해 만든 무료 매크로.

비상업적 목적, 개인 사용 목적으로는 가능하나, 암표 판매 등의 목적으로는 엄격히 사용 금지.

### 사용방법
1. 프로젝트 clone
2. chromedriver 설치
https://chromedriver.chromium.org/downloads
3. 프로젝트 내부에 `chromedriver`라는 이름으로 저장
4. `pip install -r requirement.txt`
5. config.py에 정보 입력
6. `python main.py`
